{% trans '¿Cómo Funciona?' %}
Paso 1
Debes crear un grupo, asignarle un nombre y proceder a seleccionar uno de los conectores (aplicaciones) disponible en GearPlug, por ejemplo Facebook.
Ahora proporciona la autorización necesaria para conectar esta aplicación con GearPlug. Esta autorización varía de acuerdo a la aplicación que se seleccione, en este caso, permitirnos el acceso a Facebook para enviar o recibir información relevante.
Paso 2
Paso 3
En el tercer paso, indícanos qué acción realizar desde la aplicación de origen, como registrar la información que llega de un nuevo lead, o cada vez que un usuario especiﬁco haga una comentario mencionando tu marca.
Paso 4
Paso 5
Repetir el proceso del paso 1 y 2 pero con la aplicación de destino. Es decir, la información que se adquiere debe llegar a otra aplicación para automatizar el proceso, por ejemplo escoges Facebook – Hoja de Cálculo de Google.
Indícanos qué información vamos a enviar y de qué forma. En este punto estás a muy cerca de terminar el proceso de creación de un Gear, solo debes seleccionar la información que quieres que se automatice y de qué manera, estas opciones están ligadas a las aplicaciones que escojas, una vez acabes este paso, el Gear estará listo para empezar a optimizar la recepción de datos.
¡Felicitaciones!
Haz terminado los pasos para un Gear perfecto.
